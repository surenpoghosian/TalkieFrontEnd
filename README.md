# Getting Started with Talkie

### **1.** `git clone https://github.com/surenpoghosian/TalkieFrontEnd`
### **2.** `cd TalkieFrontEnd`
### **3.** `npm install`
### **4.** `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
